Original response

If you're seeing this response, then:

  • This is your first visit and the service worker was not installed yet.
  • This is a subsequent visit and:
    • service workers are not supported by your browser.
    • you have Javascript disabled so the service worker could not be installed.
    • the service worker was blocked from installation or otherwise failed.
    • the service worker has since been forcibly unregistered.

NB: Whether the origin server was actually hit to get this response is another question entirely (due to caching etc).
